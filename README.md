# Customer Registry

## Overview

Customer Registry is a Full Stack Web Application developed as part of the SkillWallet Full Stack Developer Program. The purpose of this project is to provide a centralized system for storing, managing, and maintaining customer information securely and efficiently.

This application is being developed using modern web technologies including React.js, Node.js, Express.js, and MongoDB following the MVC (Model-View-Controller) architecture.

## Project Objectives

- Store customer information in a centralized database.
- Provide an easy-to-use interface for customer management.
- Implement CRUD (Create, Read, Update, Delete) operations.
- Ensure data security and validation.
- Follow industry-standard full-stack development practices.

## Technical Architecture

The Customer Registry project follows a three-tier architecture.

### Presentation Layer
The frontend is developed using React.js and provides a responsive user interface for interacting with customer data.

### Business Logic Layer
The backend is developed using Node.js and Express.js. It processes client requests, validates data, and communicates with the database.

### Database Layer
MongoDB is used to store customer records securely and efficiently.
